## Run it
1. Install [Node.js](https://nodejs.org/)
2. Install http-server
````bash
sudo npm i -g http-server
````
3. Go to root folder and run
````bash
http-server -c-1
````
4. Open in browser [http://127.0.0.1:8080/](http://127.0.0.1:8080/)

## Compile Sass
Install gulp
````bash
sudo npm i -g gulp
````
Go to root folder and run
````bash
npm install
gulp
````
